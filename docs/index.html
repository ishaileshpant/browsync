What browsync does

Everything you need to move between browsers without losing anything.

🔍

Universal Search

FTS5 full-text search across all bookmarks and history from every browser. Prefix matching, multi-word queries.

📥

Multi-Browser Import

Chrome, Firefox, Safari, Edge, Brave, Arc. Reads native formats directly — JSON, SQLite, plist. No export step needed.

📤

Export Anywhere

Netscape HTML (importable by all browsers), JSON, CSV. Chrome-compatible JSON format for direct import.

🖥

Terminal UI

Full ratatui TUI with bookmark tree, history list, fuzzy search, and quick-open to any browser. Vim keybindings.

🔄

Background Sync

Daemon watches browser profile directories for changes. Auto-imports on bookmark or history modification.

🔒

Auth Migration

Track saved logins across browsers. Generate migration reports showing which sites need re-login when switching.

How to use it

Every command you need, from first import to daily workflow.

CommandDescription
browsync detectAuto-detect installed browsers and available data
browsync importImport bookmarks + history from all browsers
browsync import --browser chromeImport from a specific browser
browsync search "query"Full-text search across all data
browsync export --format htmlExport to Netscape HTML / JSON / CSV
browsync open <url> --browser arcOpen a URL in any installed browser
browsync tuiLaunch interactive terminal UI
browsync statusShow database stats and sync history
browsync auth listList all saved login domains
browsync auth migrate --from chrome --to arcAuth migration report between browsers
browsyncd startStart background sync daemon
 
# First run: detect and import
$ browsync detect
  [+] Chrome   active (bookmarks, history, logins)
  [+] Safari   active (bookmarks, history)

  2 browser(s) with importable data

$ browsync import
  Importing from Chrome...
    Bookmarks: 3061 imported
    History:   5614 imported

  Database totals: 3061 bookmarks, 5614 history entries

# Search across everything
$ browsync search "rust async"
  Bookmarks (2 results):
    1. The Rust Programming Language [C]
       https://www.rust-lang.org/

# Export for another browser
$ browsync export --format html -o bookmarks.html
  Exported 3061 bookmarks to bookmarks.html
